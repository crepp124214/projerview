# projerview

> 简历上写了项目，面试时却讲不清楚？让 AI 帮你把代码变成面试底气。

projerview 会读你的项目代码，像面试官一样拆解它——生成面试手册、出题、模拟面试、甚至告诉你和目标岗位有多匹配。不是背八股文，是真正理解你写了什么。

## 它能做什么

**你以为你懂自己的项目，直到面试官问了第三个追问。**

```
你的代码 ──→ 面试手册（理解项目）
         ──→ 面试题库（预判问题）  +  八股文题库（基础知识）
         ──→ 模拟面试（面试官只看你的简历提问）
         ──→ JD 匹配（精准备战）
         ──→ 面试记录（越面越准）
```

## 流程

### 1. 读懂你的项目

projerview 像资深工程师一样阅读你的代码——抓住核心链路、设计决策和技术亮点。产出面试手册，把你可能被问到的每个角度都整理好。

- 自动识别技术栈、架构模式、核心模块
- 追踪关键业务链路，提取技术亮点
- 生成「你可以这样说...」的面试话术（支持口语自然/简洁干练/学术严谨三种风格）
- 连测试代码都不放过——边界 case 就是面试追问素材

### 1.5. 知道自己几斤几两

贴一个目标岗位的 JD，projerview 会告诉你和这个岗位有多匹配：

```
JD 要求            你的项目       面试风险
───────────────────────────────────────
Redis 缓存优化      ✅ 有实现      🟡 会被深挖
微服务架构          🟡 只有概念    🔴 大概率翻车
高并发场景          ❌ 没接触过    🔴 需要准备话术
```

海投了四五家？贴多个 JD 对比，一眼看出哪家准备最充分。

### 2. 两份题库，各有用处

**项目题库**（25-35 题）：围绕你的项目本身，8 个维度覆盖项目介绍、架构设计、技术深挖、选型对比、场景题、难点挑战、反思改进、STAR 行为题。

**八股文题库**：不是通用八股文大全——只生成你项目用到的技术，再叠加 JD 要求的技术。JD 要求但项目没用到的技术会标红提醒，那就是你需要临时补课的地方。

每道题配口语化参考答案和**高频追问速答**——直接告诉你面试官下一句会问什么、你怎么答。

### 3. 面试官只看过你的简历

这是 projerview 和其他面试工具最大的区别：**面试官不知道你的项目细节**。

模拟面试开始前，你需要提供简历文件（或让 projerview 从项目分析中反推简历版）。面试官只能看到简历上那几行字——和真实面试一模一样。

> "我看你简历上写了 Redis 缓存，命中率 85%，能说说怎么做的吗？"
>
> → 你回答了什么，面试官就追问什么。你说了布隆过滤器，他就追问误判率；你没提架构，他就主动问架构。每场面试的走向都不一样。

五种模式：**真实模拟**（面试官只看简历提问）、**专题深挖**（针对某个技术点集中追问）、**压力模式**（质疑你、打断你）、**八股文模式**（练基础知识）、**混合模式**（项目+八股文交替）。

面了真实面试？回来记录被问了什么，projerview 告诉你题库命中了多少——没命中的自动补进题库，越面越准。

## 所有命令

| 说这些就行 | 做什么 |
|-----------|--------|
| `projerview` | 自动接上次进度 |
| `projerview analyze` | 重新分析项目 |
| `projerview jd-match` | 匹配目标岗位 |
| `projerview jd-compare` | 多岗位对比 |
| `projerview question-bank` | 重新生成题库 |
| `projerview mock-interview` | 开始模拟面试 |
| `projerview record` | 记录真实面试 |
| `面试准备` / `模拟面试` / `分析 JD` | 中文也行 |

## 产出

所有文件在 `.projerview/` 目录下，自动加入 `.gitignore`，不会污染你的仓库。

## 安全

- API Key、密码、连接串等敏感信息自动脱敏
- 面试材料不进 git
- 涉及公司代码建议在个人设备上使用

## FAQ

**分析要多久？** 小项目 2-3 分钟，中型 5-8 分钟，大项目 10 分钟以上。

**支持什么语言？** Python、Java、Node.js、Go、Rust、C#、Ruby，Monorepo 也能识别。

**没有简历怎么办？** projerview 会从项目分析中自动生成简历版，但有真实简历体验更逼真。

**中途中断了？** 说"暂停"就行，下次运行自动恢复。
