# projerview

> 简历上写了项目，面试时却讲不清楚？让 AI 帮你把代码变成面试底气。

projerview 会读你的项目代码，像面试官一样拆解它——生成面试手册、出题、模拟面试、甚至告诉你和目标岗位有多匹配。不是背八股文，是真正理解你写了什么。

## 它能做什么

**你以为你懂自己的项目，直到面试官问了第三个追问。**

projerview 做的事情，本质上是帮你把"我写过这个代码"升级成"我能讲清楚为什么这样写"：

```
你的代码 ──→ 面试手册（理解项目）
         ──→ 面试题库（预判问题）
         ──→ 模拟面试（实战演练）
         ──→ JD 匹配（精准备战）
         ──→ 面试记录（越面越准）
```

## 流程

### 1. 读懂你的项目

projerview 会像一个资深工程师一样阅读你的代码——不是逐行扫描，而是抓住核心链路、设计决策和技术亮点。产出一份面试手册，把你可能被问到的每个角度都整理好了。

- 自动识别技术栈、架构模式、核心模块
- 追踪 2-3 条关键业务链路
- 提取技术亮点和设计决策
- 生成「你可以这样说...」的面试话术
- 连测试代码都不放过——边界 case 就是面试追问素材

你可以选择话术风格：**口语自然**（互联网公司）、**简洁干练**（大厂技术面）、**学术严谨**（外企/研究岗）。

### 1.5. 知道自己几斤几两

贴一个目标岗位的 JD，projerview 会告诉你：

```
JD 要求            你的项目       面试风险
───────────────────────────────────────
Redis 缓存优化      ✅ 有实现      🟡 会被深挖
微服务架构          🟡 只有概念    🔴 大概率翻车
高并发场景          ❌ 没接触过    🔴 需要准备话术
```

海投了四五家？贴多个 JD 对比一下，看看哪家你最有胜算，哪家需要临时补课。

### 2. 提前知道面试官会问什么

基于手册自动生成 25-35 道核心题，覆盖 8 个维度：

| 维度 | 典型问题 |
|------|---------|
| 项目介绍 | "介绍一下你的项目" |
| 架构设计 | "为什么用这个架构？" |
| 技术深挖 | "这个模块底层怎么实现的？" |
| 选型对比 | "为什么选 Redis 不选 Memcached？" |
| 场景题 | "并发量翻 10 倍怎么办？" |
| 难点挑战 | "遇到最难的 bug 是什么？" |
| 反思改进 | "如果重来你会怎么改？" |
| 行为面试 | "你主导过什么技术决策？"（STAR 法） |

每道题配口语化参考答案和高频追问速答。同时自动生成一份**八股文题库**——只覆盖你项目用到的技术，不是通用大全，每道题配 2-3 句速答，面试前 5 分钟能过一遍。

### 3. 先模拟，再上场

Skill 变身面试官，逐题提问、追问、打分。四种模式随你选：

- **全量模式**：按顺序过完所有项目题，首次推荐
- **专题模式**：哪个弱练哪个
- **随机模式**：每类抽 2 题，快速检验
- **压力模式**：质疑你、打断你、挑战你——提前适应真实面试的压迫感
- **八股文模式**：专门练基础知识，只问你项目用到的技术
- **混合模式**：项目题和八股文交替出题，模拟真实面试节奏

每次面试结束，你会看到自己的成绩变化曲线。面了真实面试？回来记录被问了什么，projerview 会告诉你题库命中了多少、漏了哪些——没命中的自动补进题库，越面越准。

## 所有命令

| 说这些就行 | 做什么 |
|-----------|--------|
| `projerview` | 自动接上次进度 |
| `projerview analyze` | 重新分析项目 |
| `projerview jd-match` | 匹配目标岗位 |
| `projerview jd-compare` | 多岗位对比 |
| `projerview question-bank` | 重新生成题库 |
| `projerview mock-interview` | 开始模拟面试 |
| `projerview record` | 记录真实面试 |
| `面试准备` / `模拟面试` / `分析 JD` | 中文也行 |

## 产出

## FAQ

**分析要多久？** 小项目 2-3 分钟，中型 5-8 分钟，大项目 10 分钟以上。

**支持什么语言？** Python、Java、Node.js、Go、Rust、C#、Ruby，Monorepo 也能识别。

**题库有多少题？** 核心 25-35 题，扩展不限。全量面试约 1-1.5 小时。

**中途中断了？** 说"暂停"就行，下次运行自动恢复。
